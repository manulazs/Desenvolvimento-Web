# Desenvolvimento-Web
Repositorio para armazenamento de informações, trabalhos, projetos da aula de Desenvolvimento Web da Facens
aula 1 - 
Conteúdo Programático
A disciplina abordará as seguintes áreas tecnológicas ao longo do período:
Arquitetura e Design: Definição de escopo, prototipação e fundamentos de UX/UI.
Versionamento e Infraestrutura: Controle de versões via GitHub, GitLab ou Bitbucket, além de processos de deploy e integração contínua (CI/CD).
Desenvolvimento Front-end: Lógica com JavaScript puro (Vanilla JS) e introdução a frameworks como React, Vue e Angular.
Desenvolvimento Back-end: Arquitetura MVC, APIs RESTful e frameworks de servidor.
Dados e Gestão de Conteúdo: Utilização de CMS (como WordPress e Drupal) , ferramentas de Website Builders e introdução a bancos de dados NoSQL.

Sistema de Avaliação
A composição da nota final é dividida em quatro módulos:
AC1 (15%): Prova escrita de múltipla escolha focada em teoria e aplicação de conceitos.
AC2 (30%): Desenvolvimento e apresentação de projeto, com foco em gestão de backlog e benchmarking.
Avaliação Geral (10%): Componente de avaliação contínua.
Avaliação Final (45%): Entrega de documentação completa do projeto no formato de "mini TCC".
APS: Atividades práticas supervisionadas sobre HTML e CSS para entrega via plataforma Canvas.

Atividades Iniciais Obrigatórias
Foram designadas as seguintes tarefas para execução imediata em grupos de 3 a 5 integrantes:
Repositório GitHub: Criação de diretório para a disciplina contendo um resumo da primeira aula em formato Markdown (.md).
Análise Tecnológica: Identificação das tecnologias utilizadas em três sites distintos.
Mapa Visual: Elaboração de representação colaborativa no Miro detalhando os sites analisados e suas tecnologias, com posterior salvamento da imagem no repositório.
